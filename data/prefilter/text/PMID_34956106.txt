                                             carbohydrate sources thereby influencing host health. CAZy-typing, characterizing
                                             the microbiota-associated CAZyme-coding genes within a host individual, can be a
                                             useful tool to predict carbohydrate pools that the host can metabolize, or identify
                                             which CAZyme families are underrepresented requiring supplementation via microbiota
